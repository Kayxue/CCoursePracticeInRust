# 第三次第三題
## 題目
Prime factorization is the decomposition of a positive number into a product of its prime factors, such as
90 = 2^1 × 3^2 × 5^1, or
12936 = 2^3 × 3^1 × 7^2 × 11^1.
## 範例
```
Please input a positive number: 90
90 = 2^1 * 3^2 * 5^1
```
```
Please input a positive number: 12936
12936 = 2^3 * 3^1 * 7^2 * 11^1
```
