# 第六次第二題
## 題目
Read characters from keyboard until getting a newline symbol.
Print each character (not including the newline) in square brackets, together with its following character and their ASCII codes.
## 範例
```
Alas! I am lost... @_@#
[A] 65 ==> [B] 66
[l] 108 ==> [m] 109
[a] 97 ==> [b] 98
[s] 115 ==> [t] 116
[!] 33 ==> ["] 34
[ ] 32 ==> [!] 33
[I] 73 ==> [J] 74
[ ] 32 ==> [!] 33
[a] 97 ==> [b] 98
[m] 109 ==> [n] 110
[ ] 32 ==> [!] 33
[l] 108 ==> [m] 109
[o] 111 ==> [p] 112
[s] 115 ==> [t] 116
[t] 116 ==> [u] 117
[.] 46 ==> [/] 47
[.] 46 ==> [/] 47
[.] 46 ==> [/] 47
[ ] 32 ==> [!] 33
[@] 64 ==> [A] 65
[_] 95 ==> [`] 96
[@] 64 ==> [A] 65
[#] 35 ==> [$] 36
```
